# 🐟 Cevicheria Rush

¡Bienvenido a **Cevicheria Rush**! Un juego web interactivo desarrollado completamente de forma autónoma.

## 🤖 ¿Cómo fue creado?

Este juego es un experimento único: fue **programado y diseñado al 100% utilizando el agente de IA Antigravity**, potenciado por el modelo **Gemini 3.1 Pro (High)** de Google DeepMind. 

No se escribió código manualmente; todas las mecánicas, lógicas de NPCs, sistema de órdenes, físicas y texturas fueron generadas e implementadas a través de interacciones naturales con el agente Antigravity, quien estructuró todo el proyecto web utilizando HTML5, CSS y Vanilla JavaScript puro.

## 🎮 Juega ahora

El juego está desplegado y puedes jugarlo directamente desde tu navegador gracias a GitHub Pages. 

👉 **[¡Jugar Cevicheria Rush!](https://15aozzz.github.io/cevicheria-rush/)**

## 🛠️ Tecnologías utilizadas

- **HTML5 & CSS3** para la estructura y estilos web.
- **JavaScript (ES6+)** puro, sin frameworks pesados, para la lógica principal:
  - Sistema de Órdenes (`OrderManager.js`)
  - Inteligencia de NPCs (`NPCManager.js`)
  - Controlador del Jugador (`PlayerController.js`)
  - Renderizado del Mundo y Ambiente (`World.js`, `AmbientManager.js`)
- **Git & GitHub Pages** para control de versiones y despliegue automatizado.
- **Antigravity + Gemini Pro High** como desarrollador IA autónomo.
